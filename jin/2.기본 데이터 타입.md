# 기본 타입

## Bool
var someBool: Bool = true
### swift에서는 0,1을 사용할 수 없음. true, false만 사용 가능

## Int
var someInt: Int = -100

## UInt
var someUInt: UInt = 100
### 부호가 없는 양의 정수만 사용 가능
### someUInt = someInt 불가능 
## swift는 자료형에 엄격하기 때문에 다른 데이터 타입 간에 자료 교환이 어려움

## Float
var someFloat: Float = 3.14
someDouble = 3 

## Double
var someDouble: Double = 3.14

## Character
